About This App

Purpose

Student Finance Tracker helps you manage your personal finances effectively. Track your spending, set budgets, and analyze your financial habits with interactive dashboards and advanced search capabilities.

Features

  • Real-time spending dashboard with statistics
  • Local data storage with import/export
  • Advanced regex-based transaction search
  • Multi-currency support with custom rates
  • Fully responsive mobile-first design
  • Complete accessibility compliance

Regex Patterns Used

Pattern Purpose Example
/^\S(?:.*\S)?$/ Description validation (no leading/trailing spaces) Lunch
/^(0|[1-9]\d*)(\.\d{1,2})?$/ Amount validation (numeric with 2 decimals) 12.50
/^\d{4}-(0[1-9]|1[0-2])-(0[1-9]|[12]\d|3[01])$/ Date validation (YYYY-MM-DD) 2025-09-29
/^[A-Za-z]+(?:[ -][A-Za-z]+)*$/ Category validation (letters, spaces, hyphens) Fast Food
/\b(\w+)\s+\1\b/i Duplicate word detection (advanced backreference) coffee coffee
